# BallHold'em

## Descrição

BallHold'em é um jogo de puzzle desenvolvido em JavaScript onde o objetivo é eliminar todas as peças do tabuleiro empilhando bolinhas da mesma cor.

## Objetivo do Jogo

- Mover as peças no tabuleiro estrategicamente
- Agrupar bolinhas da mesma cor em pilhas
- Eliminar todas as peças do tabuleiro completando as combinações

## Como Jogar

1. Clique em uma peça para selecioná-la
2. Mova a peça pelo tabuleiro
3. Empilhe bolinhas da mesma cor juntas
4. Quando uma pilha estiver completa, ela é automaticamente removida
5. Continue até limpar todo o tabuleiro

## Requisitos

- Navegador moderno com suporte a JavaScript
- Sem dependências externas

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (orgulhosamente Vanilla)
